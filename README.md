# 모집 게시판 (라이벌 & 동료)

라이벌/동료를 모집하는 웹 게시판. 회원가입(닉네임 로그인), 공고 작성·참여 신청, 점수 시스템,
관리자 승인·회원 관리, 랭킹, 댓글, 프로필(사진 포함) 기능을 갖춘 정적 페이지 + 공유 데이터 API 구성.

## 도커로 실행 (권장)

[Docker Desktop](https://www.docker.com/products/docker-desktop/) 설치 후 이 폴더에서:

```bash
docker compose up -d --build
```

- 접속: http://localhost:8931
- 같은 네트워크의 다른 사람: `ipconfig`로 확인한 내 IP로 `http://<내IP>:8931`
- 데이터는 `./data/data.json`에 저장되어 컨테이너를 지워도 유지됩니다.
- 중지: `docker compose down` / 로그: `docker compose logs -f`

## 도커 없이 실행 (개발용)

Node.js가 있다면:

```bash
node server.js
```

Node.js가 없는 Windows에서는 PowerShell 개발 서버(`serve.ps1`)를 사용할 수 있습니다
(단, 외부 접속을 열려면 관리자 권한으로 `netsh http add urlacl` 등록 필요).

## 계정

- 관리자 계정은 로그인 페이지 최초 접속 시 자동 생성됩니다 (닉네임: `관리자`).
- 일반 회원은 회원가입으로 이메일·닉네임·비밀번호를 등록하고, 로그인은 닉네임으로 합니다.

## 구조

| 파일 | 역할 |
|---|---|
| `index.html` | 게시판 (탭, 공고 작성/수정, 참여 신청, 댓글, 정렬) |
| `login.html` | 로그인/회원가입 |
| `profile.html` | 내 정보 (닉네임/비밀번호/프로필 사진 변경, 탈퇴) |
| `admin.html` | 관리자 (공고 승인, 회원 관리) |
| `ranking.html` | 점수 랭킹 (관리자 제외) |
| `store.js` | API 클라이언트 (세션 토큰 관리, /api/state·/api/action 호출, 화면용 미러 갱신) |
| `server.js` | Node 서버 (정적 파일 + 인증/권한 검증 API) |

## 보안 구조

- 비밀번호는 서버에서 salt+scrypt로 해싱해 저장하며 클라이언트로 절대 내려가지 않습니다.
- 로그인 시 세션 토큰(30일)을 발급하고, 모든 조회/변경 API는 토큰을 검증합니다.
- 권한은 서버가 강제합니다: 공고 수정·삭제는 작성자/관리자, 승인·점수 조정·회원 관리는 관리자만.
- 승인 대기 공고는 작성자/관리자에게만, 채팅은 참여자에게만 내려갑니다.
- 남은 개선 여지: 로그인 시도 횟수 제한(브루트포스 방어), CSRF 토큰(현재는 Bearer 헤더 방식이라 위험 낮음).
